## 📂 Featured Projects

**[🏥 Hospital Management System & Patient Portal](https://github.com/Mohammed-Alhamati/hospital-portal)**
A Laravel REST API powering three role-based portals (patient · doctor · admin). Appointment booking with **database-level double-booking prevention**, a live check-in queue, telehealth video calls (Agora), and an EHR module with **signed, time-limited document links**.

**[💰 Gold Store — E-Commerce Platform](https://github.com/Mohammed-Alhamati/Gold-Store)**
A gold & jewelry storefront with a full admin dashboard. Its core is a **live pricing engine** — real-time gold rates, per-karat (18/21/24K) calculation, hourly caching, and automatic fallback to the last known price when the upstream API fails.

**[🍽️ Restaurant Ordering Platform](https://github.com/Mohammed-Alhamati/restaurant-platform)**
A Laravel API with an animated front-end. Features a customizable add-on system, a **price/name snapshot pattern** so past orders stay accurate after menu changes, and a real-time kitchen dashboard.

> 📌 Academic & internship work (**GPMS** graduation project, **GradTrack** internship) and ML projects (employee-absence prediction, algorithmic-trading research) available on request.
